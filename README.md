# Adventure Seeker

### Prerequisites

you need to have [Node](https://nodejs.org/en/) and [MongoDB](https://docs.mongodb.com/manual/installation/) on your local machine

### Installing

Run on ios

```shell
react-native run-ios
or
expo start

```
run on andriod

```shell
react-native run-android
or
expo start
```
